# دیباچہ

### فزیکل اے آئی: مجسم انٹیلی جنس کا نیا دور

خوش آمدید! آپ ایک ایسے سفر کا آغاز کر رہے ہیں جو مصنوعی ذہانت (AI) کو فزیکل دنیا سے جوڑتا ہے۔ یہ کتاب محض تھیوری نہیں، بلکہ ایک مکمل گائیڈ ہے جو آپ کو **Humanoid Robots** بنانے اور سمجھنے میں مدد دے گی۔

### یہ کتاب کس کے لیے ہے؟

- **طلباء**: جو روبوٹکس اور AI میں اپنا کیریئر بنانا چاہتے ہیں۔
- **انجینئرز**: جو ROS 2 اور NVIDIA Isaac Sim سیکھنا چاہتے ہیں۔
- **محققین**: جو ایمباڈڈ انٹیلی جنس (Embodied Intelligence) پر کام کر رہے ہیں۔

### اس کورس میں آپ کیا سیکھیں گے؟

1. **Foundations**: فزیکل اے آئی کی بنیادیں۔
2. **ROS 2**: روبوٹس کا آپریٹنگ سسٹم۔
3. **Simulation**: ڈیجیٹل ٹوئنز اور سمولیشن۔
4. **Humanoid Engineering**: روبوٹس کی مکینیکل اور الیکٹریکل ساخت۔
